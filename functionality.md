# Whisper Push-to-Talk - Fonctionnalites

## Transcription vocale

- **Push-to-Talk** : Maintenez une touche (F2 par defaut) pour enregistrer votre voix, rel√¢chez pour lancer la transcription automatique.
- **Transcription par IA** : Le texte est retranscrit par le modele Whisper (OpenAI) avec acceleration GPU si disponible.
- **Detection automatique de la langue** : La langue parlee est detectee automatiquement, ou peut etre forcee (Francais, English, Espanol).
- **Choix du modele** : Selecteur de model
- **Gestion des modeles** : Interface dediee pour telecharger, visualiser et gerer les modeles installes.

## Collage automatique

- **Auto-paste** : Le texte transcrit est automatiquement colle dans l'application active (simulant Ctrl+V).
- **Preservation du presse-papier** : Le contenu precedent du presse-papier est sauvegarde puis restaure apres le collage.

## Coupure automatique du son

- **Auto-mute** : Le son du systeme est automatiquement coupe pendant l'enregistrement pour eviter que les bruits de l'ordinateur ne polluent la prise de son.
- **Restauration du son** : Le volume est restaure a son etat precedent des que l'enregistrement se termine.

## Raccourci personnalisable

- **Hotkey configurable** : Le raccourci Push-to-Talk peut etre modifie a tout moment (touche unique ou combinaison de touches).
- **Raccourci global** : Le raccourci fonctionne meme quand l'application n'est pas au premier plan.


## System tray

- **Icone dans la barre des taches** : L'application reste accessible depuis la zone de notification Windows.
- **Demarrage minimise** : Option pour lancer l'application directement dans le system tray sans afficher la fenetre.
- **Afficher / Masquer** : La fenetre peut etre cachee et reaffichee a tout moment depuis l'icone du tray.

## Interface

- **Theme sombre** : Interface moderne avec un theme sombre adapte a un usage prolonge.
- **Fenetre deplacable** : La fenetre peut etre deplacee librement par glisser-deposer sur la barre de titre.
- **Indicateur de statut** : Un indicateur visuel montre l'etat en cours (Pret, Enregistrement, Traitement, Termine).
- **Zone de transcription** : La derniere transcription est affichee dans une zone de texte dediee.

## Configuration persistante

- **Sauvegarde automatique** : Tous les reglages (langue, modele, raccourci, options) sont sauvegardes et restaures au prochain lancement.
